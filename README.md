# compartida
Comparte tu experiencia.  Películas, series, libros, museos, bares, restaurantes y mucho más 
MEMORIA DE PROYECTO: Automatización Gmail a GitHub 


Estado actual: Fase 1 (Manual) completada. 


Objetivo: Convertir correos específicos en documentación Markdown para un repositorio. 

Reglas de Procesamiento (Lo que la IA debe recordar): 


Búsqueda: Buscar correos en Gmail por temas (Cine, Restaurantes, etc.). 


Extracción: No olvidar nunca el emisor, la fecha del correo y la fecha del evento. 


Formato de salida: Siempre en Markdown estructurado con títulos (#), negritas (**) y listas (*). 


Almacenamiento: Los resúmenes deben ser aptos para ser pegados en archivos .md de GitHub.
